# virtuallenv
[在pycharm中使用配置好的virtualenv环境](https://blog.csdn.net/wowocpp/article/details/80786083)
```bash
# 文件夹内　创建一个虚拟环境
python2 -m virtualenv venv
# 激活虚拟环境
. venv/bin/activate
# 安装依赖
pip install ｘｘｘ
```
# Install dependencies:
```bash
pip install -r ./requirements.txt
```
